# Release notes v7.6

## `competitivePlayers` in `OptionsSpec`

`OptionsSpec` now supports a `competitivePlayers` property, allowing games to declare which players are competing against each other directly in the game options.

## Siblings animations

Items that share the same location area now animate smoothly when a sibling is added or removed, making space transitions feel natural. Siblings animations are skipped on `CreateItem` moves to avoid visual noise during setup.

## `GameErrorBoundary`

A new `GameErrorBoundary` component catches rendering errors and displays a user-friendly message instead of a blank page.

## `SetupLog`

A new `SetupLog` component allows games to display logs of things that happened automatically before the first move (e.g. initial card deals, starting player selection).

## Bug report button

A new bug report button and dialog has been added, allowing players to report issues directly from the game interface. The button is automatically hidden during tutorials.

## Export `GamePoints` components

`GamePoints` components are now exported from `react-game`, making them reusable in game-specific code.

## Zoom improvements

- When the `GameTable` zoom is enabled, the viewport now sets `maximum-scale=1` and `user-scalable=no` to prevent conflicts with native zoom
- When the `GameTable` zoom is disabled, pinch-zoom is now allowed on mobile devices

## `GridLocator` improvement

`GridLocator` now uses `countItems` for `positionDependencies` instead of a cached delta, providing more reliable repositioning.

## Improved hide / ignore logic

The hide and ignore logic on locators has been improved for more consistent behavior.

## Datadog logging improvements

- Improved Datadog configuration
- Reduced noise by filtering out network errors and fetch errors
- Added `LOGGER_LEVEL` support for console log integration with Datadog

## Other

- Upgraded `i18next`
- Close icon now renders on top of sticky headers in dialogs

## Bugfixes

- Fixed dice random number generation (upper boundary was exclusive)
- Fixed dropped items animations with rotations
