# Release notes v7.3

## TRPC migration

Complete migration from **GraphQL** to **TRPC** for the API layer: chat, playMove, undoMove, eject, giveUp, claimVictory, rematch, deckbuilding, and more.

## Translations & credits

- **S3 translations**: translation files are now served from S3 instead of being bundled in the code
- Missing translation keys are now saved automatically in production mode
- **Board game credits**: credits are now loaded from the server
- **Board game name**: the game name is now fetched from the server

## Other changes

- `GameTable.tableFontSize` is now open to modifications
- Added `TOURNAMENT` value in `GameMode`
