# Release notes v7.2

## UI improvements

- **Polyhex merge fix**: fixed an issue with polyhex tile merging
- **Portals removed**: portals are no longer used in the rendering pipeline
- **Automatic `document.lang`**: the document language is now automatically updated from the URL locale
- **Help dialog images**: better sizing of pictures inside help dialogs
- **Counter onClick**: counters now support an `onClick` event
