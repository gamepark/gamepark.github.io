# Migration to framework v7.3

## Simplify LoadingPage

The `LoadingPage` component has been simplified. Remove all unnecessary props — only the minimum is now required.

## New translation process

The translation system has changed. Translation files are no longer bundled in the code.

### Remove translation setup

- Delete the `translation.json` file from your project
- Remove the call to `setupTranslation` from your code

### Upload translations (admin)

Translations must now be uploaded by an admin through the admin interface.

If you have made changes to `translation.json` since the migration from the Google Sheet, follow this process:

1. **Delete all existing translations** for the game via the admin interface
2. **Edit your `translation.json`**: replace the `"translation"` namespace with your board game id (the game's translation namespace)
3. **Remove all `"common"` translations** from the file — they are shared across all games and must not be included
4. **Upload** the resulting JSON file through the admin interface
5. **Publish** to S3
